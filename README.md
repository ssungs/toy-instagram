# Toy-project

인스타그램 만들기

Django를 통해 인스타그램을 만들어보기

## 환경 설정

사용된 환경

```
asgiref==3.3.4
Django==3.2.4
pytz==2021.1
Pillow==8.2.0
sqlparse==0.4.1
```

requirements 를 이용한 패키지 설치

```
pip install -r requirements.txt
```
